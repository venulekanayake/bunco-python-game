# Bunco – Dice Game in Python

## Description
Bunco is a console-based dice game implemented in Python.  
This repository contains:  
1. The **interactive Bunco game** (`bunco.py`)  
2. A **simulation script** (`simulate_bunco.py`) that runs 1,000 games automatically and analyzes the results.  

It demonstrates programming fundamentals, probability analysis, and simple data visualization.

---

## Features
- Interactive console game for 2–6 players  
- Rolls dice automatically and tracks scores  
- Option to reroll selected dice (interactive game)  
- Saves game results to `scores.txt`  
- Automated simulation of 1000 games with visualization  
- Analysis of which player numbers win most frequently  

---

## How to Run

### Interactive Game
```bash
python bunco.py
```

### Simulation Script
```bash
python simulate_bunco.py
